# Baisse de loyer et taux hypothécaire de référence

Le locataire peut demander la **baisse** de son loyer lorsque les circonstances ayant justifié son fixation ont changé en sa faveur (CO art. 270a).

## Causes de baisse (CO art. 270a + OBLF art. 13)

- Baisse du **taux hypothécaire de référence**.
- Baisse de l'IPC (loyers indexés).
- Baisse durable des charges du bailleur (rares).

## Calcul de la baisse

### Sur la base du taux de référence

Chaque baisse de **0.25 point** du taux donne droit à environ **2.91 %** de baisse de loyer (asymétrie par rapport à la hausse, dont la marge est 3 % — OBLF art. 13 al. 4).

| Variation taux | % baisse | % hausse |
|----------------|----------|----------|
| 0.25 point | 2.91 % | 3.00 % |
| 0.50 point | 5.66 % | 6.09 % |
| 1.00 point | 10.78 % | 12.36 % |

### Compensation avec IPC + entretien

Si l'IPC a augmenté de X % et les frais d'entretien aussi, le bailleur peut **compenser** la baisse demandée :

```text
Baisse théorique (taux réf.) : -2.91 %
Compensation IPC 40 % de Y % : +Y × 0.4
Compensation entretien        : +Z % (selon OBLF art. 14)
Baisse effective              = max(0 ; baisse théorique − compensations)
```

## Procédure

1. **Demande écrite** du locataire au bailleur (informelle ou formelle).
2. **Calcul** annexé à la demande.
3. **Délai** : à n'importe quel moment, mais effet à la **prochaine échéance** (avec préavis identique au congé contractuel).
4. **Réponse du bailleur** : doit répondre dans un délai raisonnable (généralement 30 jours).

## Refus du bailleur

Le locataire saisit l'**autorité de conciliation** :

- Délai : 30 jours dès la réponse négative (ou son absence dans un délai raisonnable).
- Gratuité de la procédure.
- En cas d'échec : tribunal des baux.

## Stratégie de surveillance

Pour le locataire :

- Surveiller les publications trimestrielles OFL.
- Conserver le **taux de référence en vigueur lors du dernier ajustement** comme base de comparaison.
- Sur les conventions « loyer brut indexé », vérifier l'évolution simultanée IPC + taux + entretien.

Pour le bailleur (régie) :

- Anticiper les demandes de baisse en projetant les recettes futures.
- Documenter les hausses de charges (entretien, IPC, impôts) pour compensation.
- Préparer les calculs pour défendre l'absence d'obligation de baisser.

## Historique du taux de référence

L'OFL publie chaque trimestre depuis 09.2008. Évolution clé :

| Année | Taux |
|-------|------|
| 2008 | 3.50 % (introduction) |
| 2012 | 2.25 % |
| 2017 | 1.50 % |
| 2020 | 1.25 % |
| 09.2023 | 1.75 % (hausse) |
| 09.2025 | 1.50 % (baisse) |

## Pièges fréquents

- **Loyer initial** : si le loyer initial était bas (fixé sous un taux haut), tous les calculs partent de cette base.
- **Compensations cumulées** : un bailleur diligent additionne plusieurs périodes IPC + entretien depuis la dernière fixation.
- **Travaux à plus-value** : annulent les marges de baisse si intervenus depuis la dernière fixation.

## Cas particulier : indexation contractuelle (CO art. 269b)

Pour les baux **indexés** sur l'IPC (≥ 5 ans), la baisse du taux de référence n'est **pas** applicable. Seules les variations de l'IPC sont opposables.
