# Impôt sur les gains immobiliers

Impôt cantonal et communal sur les plus-values réalisées lors de la vente d'un immeuble. Bases : LHID art. 12 et lois cantonales.

## Principe (LHID art. 12)

Tous les cantons doivent prélever un impôt sur les gains immobiliers. Régimes :

| Système | Cantons |
|---------|---------|
| **Moniste** | Gain imposé séparément même pour les indépendants (ZH, BE, BS, BL, NW, JU, UR, SZ, TI) |
| **Dualiste** | Gain commercial intégré au revenu ordinaire (autres cantons : GE, VD, FR, NE, VS) |

## Calcul

```text
Gain imposable = Prix de vente − Coût d'investissement
Coût d'investissement = Prix d'achat
                      + Frais d'acquisition (notaire, RF)
                      + Travaux à plus-value (justifiés)
                      + Commissions de vente
```

## Travaux à plus-value vs entretien

| Type | Déductible du gain ? |
|------|---------------------|
| Construction extension | Oui (plus-value) |
| Aménagement combles | Oui |
| Installation piscine | Oui |
| Réfection toiture identique | Non (entretien) |
| Rénovation cuisine | Mixte selon canton |
| Isolation thermique | Souvent reconnu plus-value si non déduit du revenu |

Règle : non-cumul possible. Si déduit au revenu (LIFD art. 32), pas déductible du gain.

## Taux dégressifs selon durée de possession

Tous les cantons appliquent une **dégressivité** : plus la possession est longue, plus le taux baisse.

### Genève — barème (LCP art. 84)

| Durée de possession | Taux |
|--------------------|------|
| < 2 ans | 50 % |
| 2-4 ans | 40 % |
| 4-6 ans | 30 % |
| 6-8 ans | 20 % |
| 8-10 ans | 15 % |
| 10-25 ans | 10 % |
| ≥ 25 ans | 0 % (exonération) |

### Vaud (LMSD art. 67)

| Durée | Taux |
|-------|------|
| 0-1 an | 30 % |
| 1-2 ans | 22 % |
| 2-4 ans | 21 % |
| 4-6 ans | 19 % |
| ... décroissant ... | |
| ≥ 24 ans | 7 % |

### Zurich (StG ZH §225)

Surtaxe de **+50 %** si vente dans la 1re année, **+25 %** dans la 2e année (spéculation).

## Cas d'exonération (LHID art. 12 al. 3)

- Transfert par succession / partage successoral / donation.
- Transfert entre époux à la suite du régime matrimonial.
- Remploi de logement principal (LHID art. 12 al. 3 let. e).

### Remploi logement principal

Conditions :

- Logement vendu = résidence principale du contribuable.
- Réinvestissement dans un nouveau logement principal en Suisse.
- Délai : généralement **2 à 4 ans** (selon canton, certains 5 ans).
- Pas d'exonération définitive : seul un **report** d'imposition jusqu'à la prochaine vente.

```text
Vente résidence principale     :  CHF 1'200'000
Coût investissement initial    :  CHF   800'000
Gain                            :  CHF   400'000

Achat nouveau logement principal : CHF 1'500'000
→ Remploi total → gain reporté
   Coût investissement nouveau immeuble = 1'500'000 − 400'000 = 1'100'000
   À la prochaine vente, la base est 1'100'000 (gain latent reporté).
```

## Sur-imposition court-terme (anti-spéculation)

| Canton | Mesure |
|--------|--------|
| GE | 50 % si < 2 ans |
| VD | 30 % si < 1 an |
| ZH | Surtaxe |
| BE | Surtaxe |
| FR | Surtaxe |

## Procédure

1. Déclaration au registre foncier du canton lors de la vente.
2. Calcul par l'administration sur la base des actes notariés.
3. Décision de taxation (LP — délai habituel 30 jours pour réclamer).
4. Paiement : généralement à la charge du **vendeur**, mais **garantie hypothécaire légale** sur l'immeuble (LP art. 836).

## Imposition différée pour entreprise (LIFD art. 30, LHID art. 8)

- Remploi de biens immobilisés à l'actif d'une entreprise → report jusqu'à la prochaine aliénation, sous conditions.
- Cas typique : réinvestissement dans un autre immeuble d'exploitation.

## Particularité Genève : impôt sur les bénéfices et gains immobiliers (IBGI)

Genève impose à la fois :

- IBGI (sur le gain immobilier, exclusif des autres impôts).
- IBM (sur le bénéfice de la SA propriétaire, dualiste).
