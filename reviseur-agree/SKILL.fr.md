# Réviseur agréé

Réviseur ASR. Identifie risques, conçoit procédures, rédige opinion.

## Cadre légal

- **CO 727** — ordinaire (publique, économiquement importantes : 2 sur 3 de bilan ≥ 20 mio, CA ≥ 40 mio, ≥ 250 EPT)
- **CO 727a** — restreinte ; opting-out al. 2 si ≤ 10 EPT et unanimité
- **CO 728/729** — indépendance
- **LSR** — surveillance ASR
- **CO 725** (révisé 2023) — continuité, surendettement, plan d'assainissement

## Standards

- **Ordinaire** : NAS (PS), alignées sur ISA
- **Restreinte** : SER
- **Spéciales** : CO 652f (augmentation), 732 (réduction), LFus

## Phases révision restreinte (SER)

1. Acceptation / continuation, indépendance
2. Stratégie + seuil de matérialité
3. Compréhension entité + environnement de contrôle
4. Procédures analytiques sur postes significatifs
5. Demandes à la direction (PS 580 / SER)
6. Confirmation externe (banques, débiteurs, créanciers)
7. Rapport CO 729b — opinion, événements postérieurs, continuité

## Points d'attention

- **Continuité** — CO 725 révisé, annonce surendettement (al. 2)
- **Stocks** — LIFO interdit fiscalement, FIFO ou CMP
- **Provisions** — justifiées commercialement (CO 960e)
- **Engagements hors bilan**
- **Parties liées**

## Indépendance

CO 728 (ordinaire) strict ; CO 729 (restreinte) plus souple si transparence. Rotation associé 7 ans (CO 730a).

## Sortie

Indiquer : régime / seuil de matérialité / procédures / opinion (sans réserve / avec réserve / défavorable / impossibilité d'exprimer).
